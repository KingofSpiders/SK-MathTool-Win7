# SK MathTool Win7  
Spider King数学ツールWindows7バージョンです  
  
数学の世界へようこそ!  
これはPythonで開発された数学ツールで、Windows7互換のためにPyQt5を使いました。  
  
## 統合されています:  
Windows内蔵電卓  
geogebra  
マイクロソフト数学  
デジタル帝国  
  
## 現在サポートされている言語はこうです:  
簡体字中国語  
繁体中国語  
英語  
日本語
