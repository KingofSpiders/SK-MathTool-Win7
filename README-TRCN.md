# SK-MathTool-Win7  
Spider King數學工具 Windows7版本  
  
歡迎來到數學世界!  
這是一個用Python開發的數學工具，爲了兼容Windows7，我使用了PyQt5。
  
## 它集成了:  
Windows内置計算器  
GeoGebra  
微軟數學  
數字帝國  
  
## 它目前支持的語言:  
簡體中文  
繁體中文  
英語  
日語
