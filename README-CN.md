# SK-MathTool-Win7  
Spider King数学工具 Windows7版本  
  
欢迎来到数学世界!  
这是一个用Python开发的数学工具，为了兼容Windows7，我使用了PyQt5。
  
## 它集成了:  
Windows内置计算器  
GeoGebra  
微软数学  
数字帝国  
  
## 它目前支持的语言:  
简体中文  
繁体中文  
英语  
日语
